### Pre-requisites
* Node.js ( suggested v4.4.5)
* CouchDB


## Install Hoodie Cli
[hood.ie Docs](http://docs.hood.ie/en/start/)
```
$ npm install -g hoodie-cli
```

## Create new backend app
```
$ hoodie new testapp
```

## Run
```
$ npm start -- --custom-ports 6004,6005,6006
```
