## Pre-requisites
* Nodejs ( suggested v4.4.5)

## Setup:

```
$ cd other/tools/
$ npm install
$ npm install -g gulp gulper
$ bower install
```

## Run:
```
$ gulper
```

or if you're on windows7 [gulp doesn't run on Win7](https://github.com/anatoo/gulper/issues/2):

```
$ gulp
```
